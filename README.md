# Global Sales EDA

EDA of a global retail company (online + offline).

## Key findings
- Europe generates **94.2% of revenue** and **94.7% of profit** (high geographic concentration).
- Core categories: Office Supplies, Household, Cosmetics.
- Online and offline channels contribute almost equally.
- Median shipping time ~25 days; long-tail delays exist.
- No strong negative relationship between shipping time and profit.

## How to run
pip install -r requirements.txt  
Open Global_Sales_EDA.ipynb
